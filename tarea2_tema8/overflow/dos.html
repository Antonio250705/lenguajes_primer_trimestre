Overflow


Controla el comportamiento de un contenedor cuando su contenido es más grande que el contenedor mismo y se desborda

  • visible: Este es el valor por defecto. El contenido que se desborda simplemente se muestra fuera del contenedor, sobresaliendo visualmente.
  • hidden: El contenido que se desborda está oculto y no es visible. Esto puede ser útil cuando no quieres que el contenido que sobresale sea visible.
  • scroll: agrega una barra de desplazamiento al contenedor, permitiendo al usuario desplazarse para ver el contenido que está fuera de la vista.
  • auto: Similar a scroll, pero se agregarán barras de desplazamiento solo cuando sea necesario. Si el contenido se ajusta dentro del contenedor, no se mostrarán barras de desplazamiento.
